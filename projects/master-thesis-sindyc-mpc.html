Master Thesis: SINDYc for Nonlinear MPC on evoBot

Nonlinear system identification and predictive control using Sparse Identification of Nonlinear Dynamics with control inputs.

Read web summaryReport upload guide

Executive summary

The thesis focused on identifying compact nonlinear dynamics from experimental or simulated system data and evaluating the identified model inside a predictive control workflow.

The core engineering question was how to move from observed input-output behavior to a control-oriented model that can support Model Predictive Control without relying only on hand-derived equations.

The work connected data-driven modeling, sparse regression, controller design, and validation on evoBot, with the project conducted externally with Fraunhofer IML.

Problem framing

Mobile robotic systems can be difficult to model accurately when dynamics are nonlinear, coupled, or affected by changing operating conditions. A useful controller needs a model that is accurate enough for prediction, compact enough for optimization, and transparent enough for engineering review.

Engineering approach

  • Collected or structured time-series data covering system states, inputs, and response behavior.
  • Built a sparse candidate library of nonlinear terms that could explain the observed dynamics.
  • Used SINDYc to identify governing equations with control inputs.
  • Evaluated whether the identified model could be used inside an MPC workflow.
  • Compared tracking behavior, prediction quality, and controller feasibility against project requirements.

Outcomes and value

  • Created a clear pipeline from measured data to sparse nonlinear model discovery.
  • Demonstrated how identified models can support control-oriented prediction.
  • Built a thesis narrative combining robotics, control theory, and data-driven system identification.
  • Thesis grade: 1.7.
